Difference between function level scope and block level scope?

A function scoped variable means that the variable defined within a function will not accessible from outside the function. on the other hand, A block scoped variable means that the variable defined within a block will not be accessible from outside the block. Variables declared with var, let and const are quite similar when declared inside a function. let & const are block scope except var.

What is an event loop in JavaScript?

The event loop is the secret behind JavaScript’s asynchronous programming. JS executes all operations on a single thread, but using a few smart data structures, it gives us the illusion of multi-threading.
